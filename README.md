# JOKENPO

A little unique Jokenpo game where the player can select from a wide range of activities. <br>
Made using Flutter.

## Actions

- Rock
- Paper
- Scissor
- Fire
- Dragon
- Water
- Lightning
- Wind

## Running the application

In order to run the app you have to add the dependencies  
Run the command:

```
flutter pub get
```
